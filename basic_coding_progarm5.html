PYTHON CODING

6.1 การใช้ if, else
 คือการใช้ตรรกรร ในการแยกต่างๆเช่น

คือ กำหนดให้ x มีค่าเท่ากับ 12
ถ้า x มี่ค่า มากว่าหรือเท่ากับ 10 ให้ปรื้น "pass"

6.2 if, else

คือ กำหนดให้ x มีค่าเท่ากับ 40 ถ้า x มีค่ามากกว่าหรือเท่ากับ 50 ปริ้น "pass" นอกจากนั้นให้ปริ้น "Not pass"
